About Me

I'm Brian Williams, a Computer Science student with a passion for game development, gameplay programming, and interactive systems. I enjoy building mechanics, user interfaces, visual effects, and gameplay features while continuously expanding my technical skills.

My Journey Into Game Development

Starting with Unity

My game development journey began with Unity. Through personal projects and prototypes, I learned the fundamentals of game development including player movement, collision systems, UI design, scene management, game loops, and C# scripting.

Unity gave me the opportunity to understand how games are structured and helped build the foundation that I continue to use today.

Transitioning to Unreal Engine 5

As my interests shifted toward gameplay systems, visual effects, and larger-scale development, I began learning Unreal Engine 5.

Unreal's combination of C++, Blueprints, Niagara, and powerful built-in systems provided the tools I was looking for to create more advanced gameplay experiences.

Since making the transition, I've focused on building inventory systems, interaction frameworks, gameplay UI, visual effects, combat mechanics, and reusable gameplay architecture.
